## **Section 5: Control Flow**

### **Conditional Statements**
* `if/else`
* `switch/case`
* conditional operator `a ? b : c`

```csharp
// if/else example:
if (condition) 
{
    someCode // if you have one line of code, then no curly braces are needed
}
else if (condition)
{
    someMoreCode
}
else 
{
    evenMoreCode
}

// switch/case example:
switch (role)
{
    case Role.Admin:
        ...
        break;
    case Role.Moderator:
        ...
        break;
    default:
        ...
        break;
}
```
